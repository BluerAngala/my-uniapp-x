# uni-app x 练手项目

基于 uni-app x 框架开发的练手项目，包含常用 UI 组件和页面模板。

## 技术栈

- uni-app x
- UTS（强类型 TypeScript）
- uniCloud

## 功能规划

- [ ] 常用 UI 组件
- [ ] 常用页面模板
- [ ] 最佳实践示例

## 开发环境

使用 HBuilderX 进行开发和调试。

## 目录结构

```
├── pages/          # 页面文件
├── components/     # 公共组件
├── static/         # 静态资源
├── uni_modules/    # uni-app 插件
└── uniCloud-alipay/ # 云函数
```
